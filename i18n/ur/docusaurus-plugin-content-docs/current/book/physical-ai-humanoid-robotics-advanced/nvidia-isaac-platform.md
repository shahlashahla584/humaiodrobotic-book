# NVIDIA Isaac پلیٹ فارم

## سیکھنے کے مقاصد
اس فصل کے آخر تک، طلباء مندرجہ ذیل کر سکیں گے:
- NVIDIA Isaac پلیٹ فارم کی مجموعی معماری اور اجزاء کی وضاحت کریں۔
- بلند معیار کی روبوٹکس شبیہ سازی میں Isaac Sim کے کردار کو سمجھیں۔
- وضاحت کریں کہ Isaac Sim میں مصنوعی ڈیٹا جنریشن AI ماڈل تربیت کو کیسے سہارا دیتا ہے۔
- ROS 2 کو Isaac Sim کے ساتھ روبوٹ کنٹرول کے لیے ضم کرنے کے عمل کو بیان کریں۔
- Isaac کا استعمال کرتے ہوئے GPU تیز روبوٹکس ترقی کے فوائد پر تبادلہ خیال کریں۔

## NVIDIA Isaac کا تعارف
NVIDIA Isaac مصنوعی ذہانت سے محرک روبوٹس کی ترقی، شبیہ سازی، اور تعیناتی کو تیز کرنے کے لیے ایک جامع پلیٹ فارم ہے۔ یہ سافٹ ویئر ٹولز، SDKs، اور ہارڈ ویئر تیزی کا ایک مجموعہ ہے، جو احساس، مینوپولیشن، اور نیویگیشن تک کے روبوٹکس کے پیچیدہ چیلنجز کو حل کرنے کے لیے ڈیزائن کیا گیا ہے۔ اس کے مرکز میں، Isaac GPU کمپیوٹنگ اور AI میں NVIDIA کی مہارت کا فائدہ اٹھاتا ہے۔

### Isaac پلیٹ فارم کے کلیدی اجزاء
1.  **Isaac Sim:** NVIDIA Omniverse پر تعمیر شدہ اسکیل ایبل روبوٹکس شبیہ سازی ایپلی کیشن، جو بلند معیار، جسمانی طور پر درست مجازی ماحول فراہم کرتا ہے۔
2.  **Isaac SDK:** احساس، نیویگیشن، اور مینوپولیشن کے لیے روبوٹکس الگورتھم، فریم ورکس، اور ٹولز کا مجموعہ۔
3.  **جیٹسن پلیٹ فارم:** جسمانی روبوٹس پر AI کو ایج پر تعینات کرنے کے لیے ایمبیڈڈ کمپیوٹنگ بورڈز۔
4.  **Omniverse:** 3D ڈیزائن تعاون اور شبیہ سازی کے لیے ایک پلیٹ فارم، Isaac Sim کے لیے بنیادی ٹیکنالوجی فراہم کرتا ہے۔

## Isaac Sim: جدید روبوٹکس شبیہ سازی
Isaac Sim Isaac پلیٹ فارم کا ایک اہم جزو ہے، جو تصویری طور پر حقیقی اور جسمانی طور پر درست شبیہ سازی کا ماحول فراہم کرتا ہے۔ یہ NVIDIA Omniverse پر تعمیر کیا گیا ہے، جو حقیقی وقت 3D تعاون اور شبیہ سازی کو فعال بناتا ہے۔

### Isaac Sim کی خصوصیات
- **اعلی معیار کی فزکس:** NVIDIA PhysX 5 کی طاقت سے متحرک، جو جامد اجسام، مائع، اور تبدیل ہونے والے مواد کی درست شبیہ سازی کو یقینی بناتا ہے۔
- **فوٹو ریلزم رینڈرنگ:** گہری سیکھنے والے ماڈلز کی تربیت کے لیے ضروری، مصنوعی ڈیٹا جنریٹ کرنے کے لیے جدید رینڈرنگ کی صلاحیتیں جو حقیقی دنیا کی تصاویر سے قریبی مماثلت رکھتا ہو۔
- **مصنوعی ڈیٹا جنریشن (SDG):** شبیہ سازی سے متنوع اور نشان زد سیٹ (جیسے، RGB، گہرائی، سیگمینٹیشن ماسک، باؤنڈنگ باکس) کی خودکار تیاری۔ یہ حقیقی دنیا کا بڑا مقدار میں ڈیٹا حاصل کرنے کے چیلنج کو دوسری طرف دیکھتا ہے۔
  *ڈائریم: Isaac Sim میں مصنوعی ڈیٹا جنریشن کا ورک فلو*
- **ROS 2 انضمام:** ROS 2 کے ساتھ بے داغ ربط، جو ڈیولپرز کو شبیہ سازی شدہ ماحول کے اندر موجود ROS 2 پر مبنی کنٹرول سٹیکس اور ٹولز کو استعمال کرنے کی اجازت دیتا ہے۔
- **ریفورسمنٹ لرننگ (RL) انضمام:** بلند پیرالائزیشن والے شبیہ سازی ماحول میں RL ایجنٹس کو تربیت دینے کے لیے ٹولز اور API (جیسے، `OmniIsaacGymEnvs`) فراہم کرتا ہے۔

### مثال: Isaac Sim میں روبوٹ کو اسپون کرنا (صوری Python اسکرپٹ)
Isaac Sim کو Python کا استعمال کرتے ہوئے اسکرپٹ کیا جا سکتا ہے، Omniverse Kit API کے ساتھ تعامل کرتے ہوئے۔

```python
# Isaac Sim میں روبوٹ کو اسپون کرنے کے لیے تصوراتی Python اسکرپٹ
from omni.isaac.kit import SimulationApp

# Isaac Sim کو لانچ کریں (اسکرپٹنگ کے لیے ہیڈ لیس موڈ)
kit = SimulationApp({"headless": False})

import omni.isaac.core as ic
from omni.isaac.core.utils.nucleus import get_assets_root_path
from omni.isaac.core.utils.types import ArticulationAction

# Isaac Sim دنیا کو شروع کریں
world = ic.World(stage_units_in_meters=1.0)
scene = world.scene

# اثاثوں کا روٹ راستہ حاصل کریں (جہاں بہت سارے مثالی روبوٹس واقع ہیں)
assets_root_path = get_assets_root_path()
if assets_root_path is None:
    print("انتباہ: Isaac Sim اثاثوں کا روٹ راستہ نہیں ملا۔")
    kit.shutdown()
    exit()

# USD روبوٹ ماڈل کا راستہ تعریف کریں (جیسے، Franka Emika Panda arm)
robot_usd_path = assets_root_path + "/Isaac/Robots/Franka/franka_alt_fingers.usd"

# منظر میں روبوٹ شامل کریں
my_robot = scene.add(ic.articulations.Articulation(
    prim_path="/World/Franka",
    name="franka_robot",
    usd_path=robot_usd_path,
    position=ic.utils.prims.get_prim_at_path(robot_usd_path).GetAttribute("approx_center_y").Get() # approx_center_y حاصل کریں
))

# تبدیلیاں لاگو کرنے کے لیے دنیا کو ری سیٹ کریں
world.reset()

# چند قدم کے لیے شبیہ سازی کریں (صوری کنٹرول لوپ)
for i in range(100):
    world.step(render=True)
    if i == 50:
        # مثال: ایک بے ترتیب جوڑ کمانڈ لاگو کریں
        num_dof = my_robot.num_dof
        joint_effort_commands = ic.utils.numpy_extensions.random_sample(shape=(num_dof,))
        my_robot.apply_action(ArticulationAction(joint_effort_commands=joint_effort_commands))

kit.shutdown()
```

## مصنوعی ڈیٹا جنریشن (SDG)
SDG احساس کے ماڈلز کی تربیت میں ایک نقطہ نظر کی تبدیلی ہے۔ حقیقی دنیا کی تصاویر کو دشواری سے جمع کرنے اور نشان زد کرنے کے بجائے، Isaac Sim بڑی مقدار میں متنوع ڈیٹا کو مکمل بنیادی حقیقت والے لیبلز کے ساتھ تیار کر سکتا ہے۔ یہ حقیقی دنیا کے ڈیٹا سیٹ میں ڈیٹا کی کمی اور رجحان کے مسائل کو دور کرنے میں مدد کرتا ہے۔

- **ڈومین رینڈمائزیشن:** شبیہ سازی میں پیرامیٹرز (جیسے، ٹیکسچرز، لائٹنگ، اشیاء کی پوزیشنز) کو بے ترتیب کرنا تربیت شدہ ماڈلز کی حقیقی دنیا کے منظر ناموں تک عام بوداری کو بہتر بنانے کے لیے۔
- **لیبلنگ:** باؤنڈنگ باکس، مثال سیگمینٹیشن، گہرائی کے نکشے، اور نارمل نکشے جیسے لیبلز کی خودکار تیاری، جن کو حقیقی دنیا میں دستی طور پر حاصل کرنا مشکل یا ناممکن ہے۔

## ROS 2 اور Isaac Sim انضمام
Isaac Sim ROS 2 برج فراہم کرتا ہے، جو ڈیولپرز کو شبیہ سازی سے سینسر ڈیٹا کو ROS 2 ٹوپکس پر شائع کرنے اور ROS 2 کے کمانڈ ٹوپکس کو سبسکرائب کرنے کی اجازت دیتا ہے تاکہ شبیہ شدہ روبوٹس کو کنٹرول کیا جا سکے۔ یہ بلند معیار کے سیمیولیٹر کے ساتھ آشنا ROS 2 نیویگیشن، مینوپولیشن، اور احساس کے سٹیکس کو استعمال کرنے کی اجازت دیتا ہے۔

- **ROS 2 برج:** Isaac Sim (Omniverse) اور ROS 2 ٹوپکس/سروسز/ایکشنز کے درمیان ڈیٹا ایکسچینج کو آسان بناتا ہے۔
- **روبوٹ کی تفصیل:** URDF/USD ماڈل Isaac Sim میں لوڈ کیے جا سکتے ہیں اور ROS 2 `robot_state_publisher` کے ساتھ مطابقت پذیر کیے جا سکتے ہیں۔

## GPU تیز روبوٹکس ترقی
NVIDIA Isaac پورے روبوٹکس پائپ لائن میں GPU کی طاقت کا فائدہ اٹھاتا ہے:
- **شبیہ سازی:** حقیقت سے زیادہ تیز شبیہ سازی کے لیے پیرالائزیشن فزکس اور رینڈرنگ۔
- **AI تربیت:** احساس اور کنٹرول کے لیے گہری سیکھنے والے ماڈلز کی تربیت کو تیز کرنا۔
- **انفرینس:** روبوٹ پر AI ماڈلز کی تیز انجام دہی (جیسے، جیٹسن ڈیوائسز پر)۔
- **روبوٹکس الگورتھم:** SLAM، موشن منصوبہ بندی، اور انورس کنیمیٹکس جیسے کمپیوٹیشنل طور پر کثیر کوششوں والے الگورتھم کے GPU اجراء۔

یہ تیزی ترقی کے سائیکل کو نمایاں طور پر تیز کرتی ہے، زیادہ تکرار اور زیادہ پیچیدہ AI ماڈلز کی اجازت دیتی ہے۔

## حقیقی دنیا کی مثال: NVIDIA کا پروجیکٹ GR00T اور OS0
NVIDIA کا حالیہ اعلان پروجیکٹ GR00T (جنرلیسٹ روبوٹ 00 ٹیکنالوجی) اور ہیومنوائڈ روبوٹکس کے لیے نیا جیٹسن تھور Isaac پلیٹ فارم کے اختتام کی مثال ہے۔ GR00T ہیومنوائڈ روبوٹس کے لیے ایک فاؤنڈیشن ماڈل ہے، جو انہیں قدرتی زبان کو سمجھنے اور انسانی حرکات کو نقل کرنے کے قابل بناتا ہے۔ یہ Isaac پلیٹ فارم کے ذریعے فراہم کردہ جدید شبیہ سازی، مصنوعی ڈیٹا، اور طاقتور انفرینس کی صلاحیتوں پر بہت زیادہ انحصار کرتا ہے۔
